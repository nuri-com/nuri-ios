# Start KYC

> 🚧
>
> ### Starting Tier 2 KYC
>
> Starting 2025, a minimum deposit value may be applied for users of your application in order for them to be able to start Tier 2 KYC.
>
> This is in order to have a smoother user experience as Tier 2 KYC verifications are manually processed due to the varying nature of documents that users can upload for Source of Funds (SOF) checks. Users who have little to no transaction history do not warrant a Tier 2 SOF check and hence may not be eligible to start Tier 2 KYC, saving your application the cost of a Tier 2 check and a better user experience.
>
> If a deposit threshold is configured for your application, please pay careful attention to the error code `33003` with the message `{XX}_EUR_DEPOSIT_THRESHOLD_NOT_MET` where the XX represents the value in Euros to meet the deposit threshold conditions to start Tier 2 KYC.
