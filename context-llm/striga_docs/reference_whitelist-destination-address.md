# Whitelist Destination Address

> 🚧
>
> ### IMPORTANT
>
> When sending money to a USDC address on the Polygon network, it is crucial to whitelist the destination address specifically for USDC POLYGON. Ensure that you use the currency identifier as `USDC_POLYGON` and network as `POLYGON`.
>
> This step is essential for the successful processing of your transaction on the Polygon network.
>
> For more information please refer to this [document](https://docs.striga.com/reference/moving-money-around-1#blockchain-transactions)
