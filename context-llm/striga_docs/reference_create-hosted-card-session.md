# Start Hosted Card Session

> ❗️
>
> ### User IP Address
>
> Ensure that the IP address filled above when starting a session is the real IP address of the end user device where the session is being started. Your application is responsible for ensuring that a hosted card session is started only by the end user and that your frontend relays the correct IP address in this API request.
>
> Passing an incorrect/arbitrary IP address will deteriorate the user experience.
