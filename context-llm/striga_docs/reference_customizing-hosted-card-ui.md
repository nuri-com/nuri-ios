# Customizing the Hosted Cards User Interface

Below are example screenshots of the hosted card screens with annotations indicating which text elements are classified as heading or body. These markings help illustrate where your custom font and color styles will be applied. Note that the styles themselves are not reflected in the screenshots.

To customize the fields in the following screenshots, simply use the set/get custom CSS APIs documented in the following endpoints.

Got suggestions on additional fields you would like customization control over? Get in touch at [[email protected]](/cdn-cgi/l/email-protection#aedddbdedec1dcdaeedddadcc7c9cf80cdc1c3)
