# Business Wallets & Transactions

The following section contains API endpoints to manage accounts/wallets for users onboarded using KYB, i.e. businesses or legal entities.

Functionally consumer and business endpoints are nearly identical however function internally differently with a different set of rules/logic for checking limits, monitoring transactions etc. Use the set of APIs relevant for your business, or both if you wish.
