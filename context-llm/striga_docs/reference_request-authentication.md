# Getting Started with Striga

To get started, signup for a developer account on the [sandbox](https://portal.striga.com) and obtain credentials to make requests to the Striga APIs.

Enter your company details, select the currencies you want to enable and the type of application you are building. Once done, navigate to the "API Credentials" section through the dashboard, to get your access credentials as shown below.

## Crypto & Banking API Credentials

![](https://files.readme.io/530fd36-API-postman.gif)

Below your credentials, visible is a short code snippet made to Striga's `/ping` endpoint using your credentials to test the calculation of your Authorization header and validate your key.
