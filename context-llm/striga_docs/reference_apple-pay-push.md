# Apple Pay Push Provisioning

> 🚧
>
> ### Apple Pay Integration
>
> Please note that Apple Pay integration, involves a thorough testing/launch process co-ordinated directly between your legal entity and Apple's European Test Houses. Striga and it's issuing partners can guide you through the process, but this API is only available on production for use after your application has been approved for Apple Pay.
